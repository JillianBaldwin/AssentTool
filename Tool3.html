TOOL THREE: Establishing Participation in the Program

Purpose: Genuine assent is not present. The purpose of this tool is to establish program participation from a learner to assess for genuine assent.

Directions: Given the learner and program from Tool 1, answer the following questions.

1. Is the learner engaging in the programmed opt-out response instead of engaging in the program?

You answered Yes!

Proceed to Question 2.

2. Does the target behavior happen under any conditions (e.g., different instructors, locations, etc.)?

You answered No!

The learner is engaging in another opt-out response.

  • Make the current opt-out response easier (e.g., using a card instead of a vocal response).
  • Teach a different opt-out response.

Select and implement necessary changes to the opt-out response. Once they are engaging in a programmed opt-out response, proceed to Question 2.

2. Does the target behavior happen under any conditions (e.g., different instructors, locations, etc.)?

You answered Yes!

Consider the following:

  • Transfer procedures
  • Assess the reinforcer (is engaging in the response worth it?)
  • Analyze the antecedent context
    • Try the program in a different environment
    • Try the program in a different location
    • Try the program with different materials
    • Try the program with a different instructor
  • Add an additional alternative behavior.

You answered No!

Consider the following:

  • Make the target behavior simpler and shape approximations toward the target behavior
  • Assess the reinforcer (is engaging in the response worth it?)
  • Check for missing pre-requisite behaviors
  • Target a pre-requisite behavior
  • Analyze the antecedent context
    • Try the program in a different environment
    • Try the program in a different location
    • Try the program with different materials
    • Try the program with a different instructor
