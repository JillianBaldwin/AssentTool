TOOL TWO: Making Shift Happen

Purpose: Genuine assent is present. This tool will provide support to shift from the alternative response to the specified behavior. 

Directions: Given the learner and program from Tool 1, answer the following question. 

1. Has the specified behavior ever occurred, or does it currently occur, under any conditions (e.g., different instructors, locations, etc.)?

 

You answered Yes!

Consider the following: 

  1. Transfer procedures 

  2. Assess the reinforcer (is engaging in the response worth it?) 

  3. Analyze the antecedent context 

    • Try the program in a different environment 
    • Try the program in a different location 
    • Try the program with different materials 
    • Try the program with a different instructor

You answered No!

Consider the following: 

  1. Make the target behavior simpler and shape approximations toward the specified behavior 

  1. Assess the reinforcer (is engaging in the response worth it?) 

  1. Check for missing pre-requisite behaviors  

  1. Target a pre-requisite behavior 

  1. Analyze the antecedent context 
  • Try the program in a different environment 
  • Try the program in a different location 
  • Try the program with different materials 
  • Try the program with a different instructor 

Select the change you will make from the options above and return to Tool 1.
